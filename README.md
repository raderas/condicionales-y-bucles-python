# Estructuras condicionales y bucles en python
Introducción a estructuras condicionales y bucles en python.
Uso de instrucciones **if**, **else**, **elif**, **for**, y **while**

# Objetivo de aprendizaje
Comprender la sintaxis para definir y ejecutar sentencias de control de flujo condicionales y bucles. Mencionando casos de ejemplo para cada uno o reconociendo a partir de una descripción, cuál es la instrucción que se debería utilizar para cumplir una tarea de programación.

# Evidencia de aprendizaje esperada
Se presentan ejemplos de código adicionalmente con analogías del mundo real para que el alumno identifique en qué caso se debe utilizar cada una de las sentencias de control de flujo y pueda decidir cómo y de qué manera utilizarlas al ser presentado con una problemática de código o del mundo real donde se muestre una toma de decisión basado en condiciones *(if, elif, else)*, la ejecución de una tarea repetitiva un número específico de veces *(for)* o repetir una tarea basado en una condición *(while)*.

## Archivos de código presentados

|Archivo|Contenido|
|---|---|
|*ejemplo-if-1.py*|Sentencia condicional *if/else* simple|
|*ejemplo-if-2.py*|Sentencia condicional *if/elif/else* para evaluación de condiciones múltiples|
|*ejemplo-for-1-py*|Bucle *for* para repetir una tarea un número determinado de veces.|
|*ejemplo-while.py*|Bucle *while* para repetir una tarea mientras una condición sea cierta.|


# Criterio de evaluación
- Identificar la instrucción a utilizar partiendo de la descripción de una tarea a realizar en código
- Modificar el código de los ejemplos cambiando la definición de condiciones o variables de control para verificar cómo cambia la ejecución del programa.
- Partiendo de la definición de un caso, escribir el código que lo realiza utilizando sentencias condicionales o bucles.
